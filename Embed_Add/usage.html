How to Add Embed

  • Go to the Episode List in the admin panel.
  • Select the episode you want to add an embed to.
  • Click Add Embed or Edit Embed.
  • Paste the embed code (iframe or video link) into the provided field.
  • Choose the embed type (e.g., VidSrc, 2Anime, YouTube, etc.) if required.
  • Set the embed server order and fansub if needed.
  • Click Save to attach the embed to the episode.

Supported Embed Types

  • VidSrc - vidsrc.to
  • 2Anime - animeapi.skin
  • YouTube - youtube.com
  • and more...

Notes

  • Embed code must be a valid iframe or direct video link.
  • Server order affects the priority of the embed for the episode.
  • Fansub selection is optional but recommended for proper sorting.
