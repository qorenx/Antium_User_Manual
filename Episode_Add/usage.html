How to Add Episode

  • Go to the Episode Add section in the admin panel.
  • Select the anime you want to add episodes to.
  • Enter episode number, title, and other details (raw, sub, dub, filler, recap, etc.).
  • For batch add, specify the episode range.
  • Click Save to add the episode(s).

Notes

  • Episode number must be unique for each anime.
  • Use batch add for adding multiple episodes at once.
  • Fill in all required fields for each episode.
