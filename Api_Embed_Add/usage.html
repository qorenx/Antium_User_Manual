API Embed Usage Guide

This guide explains how to use different API sources for embedding anime episodes. For each API, you will find what to enter as api_input, how to find it, and how the system uses it. Please follow the instructions and examples for each API below.

API-1 (VidSrc)

  • Source Site: https://vidsrc.to/
Required api_input: tt2560140|1
Format: [IMDB_ID]|[SEASON_NUMBER]
  • How to find api_input:
    1. Go to the anime's IMDb page (e.g., https://www.imdb.com/title/tt2560140/).
    2. Extract the IMDb ID from the URL (e.g., tt2560140).
    3. Determine the season number you want (e.g., 1 for season 1).
    4. Combine as tt2560140|1.
  • Example api_input: tt31131295|2 (for IMDb ID tt31131295, season 2)
  • How the system uses api_input: The system generates embed URLs for each episode in the selected range by appending the episode number (e.g., .../1, .../2, etc.).
Example:
  • api_input: tt31131295|1
  • Episode Range: 1-12
  • Generated embed URLs: .../1, .../2, ..., .../12
Notes:
- Always use the IMDb ID and season number separated by |.
- If you select a single episode (e.g., 11-11), only that episode will be generated.

API-2 (2Anime)

  • Source Site: https://animeapi.skin/
Required api_input: tokidoki-bosotto-russia-go-de-dereru-tonari-no-alya-san
Format: [anime-title-in-slug-format]
  • How to find api_input:
    1. Go to the anime's MyAnimeList page (e.g., https://myanimelist.net/anime/54744/Tokidoki_Bosotto_Russia-go_de_Dereru_Tonari_no_Alya-san).
    2. Take the main title and convert it to a slug (lowercase, spaces and special characters replaced with hyphens).
    3. Example: Tokidoki Bosotto Russia-go de Dereru Tonari no Alya-san â†’ tokidoki-bosotto-russia-go-de-dereru-tonari-no-alya-san
  • Example api_input: attack-on-titan
  • How the system uses api_input: The system generates episode URLs like tokidoki-bosotto-russia-go-de-dereru-tonari-no-alya-san-episode-1, ...-episode-2, etc., for the selected range.
Example:
  • api_input: tokidoki-bosotto-russia-go-de-dereru-tonari-no-alya-san
  • Episode Range: 1-12
  • Generated URLs: tokidoki-bosotto-russia-go-de-dereru-tonari-no-alya-san-episode-1, ..., ...-episode-12
Notes:
- The api_input must be the slugified anime title.
- If you select only one episode (e.g., 11-11), only that episode will be generated.

API-3 (AnimePahe)

  • Source: Anime Mapper (GitHub)
Required api_input: 162804
Format: [AniList_ID]
  • How to find api_input:
    1. Go to the anime's AniList page (e.g., https://anilist.co/anime/162804/Tokidoki-Bosotto-Russiago-de-Dereru-Tonari-no-Alyasan/).
    2. Extract the AniList ID from the URL (e.g., 162804).
  • Example api_input: 11061
  • How the system uses api_input: The system generates URLs like API_URL/animepahe/hls/162804/1, .../2, etc., for the selected range.
Example:
  • api_input: 162804
  • Episode Range: 1-12
  • Generated URLs: API_URL/animepahe/hls/162804/1, ..., .../12
Notes:
- The api_input must be the AniList ID.
- The episode number is appended for each episode in the range.

API-4 (AniWatch / HiAnime)

  • Source Site: https://hianime.to/
  • API: AniWatch API (GitHub)
Required api_input: alya-sometimes-hides-her-feelings-in-russian-19254
Format: [anime-slug-from-hianime-url]
  • How to find api_input:
    1. Go to the anime's HiAnime page (e.g., https://hianime.to/alya-sometimes-hides-her-feelings-in-russian-19254).
    2. Copy the slug part after the last slash (e.g., alya-sometimes-hides-her-feelings-in-russian-19254).
  • Example api_input: one-piece-100
  • How the system uses api_input: The system searches for and adds all episodes in the selected range using the slug.
Example:
  • api_input: alya-sometimes-hides-her-feelings-in-russian-19254
  • Episode Range: 1-12
  • Result: All episodes from 1 to 12 will be added using the API.
Notes:
- The api_input must be the anime slug from the HiAnime URL.
- If you select only one episode (e.g., 11-11), only that episode will be added.

Notes on Episode Range

  • If you select a range (e.g., 1-12), the system will generate and add all episodes in that range.
  • If you select a single episode (e.g., 11-11), only that episode will be added.
  • Each API has specific requirements for api_input. Always use the correct value as described above.
